Welcome to Kaazing WebSocket Gateway Release 4.0



=============================================================================

   Kaazing WebSocket Gateway is an HTML5-compliant WebSocket server, with
   emulation to support legacy browsers and mobile devices.

Environment
=============================================================================

Java Requirements

    * Java Developer Kit (JDK) or Java Runtime Environment (JRE)
      Java 7 (version 1.7.0_21) or higher
    * The JAVA_HOME environment variable must be set to the directory where 
      the JDK is installed, for example C:\Program Files\Java\jdk1.7.0_21
    * Note:
      * For information on installing JDK, see Oracle's Java SE documentation:
        http://download.oracle.com/javase/.

Operating System
    * Windows: Windows 8 (including Windows Metro), Windows 7,
			   Windows Vista, Windows XP SP3, and Windows Server 2008/2012
    * Unix: Ubuntu Linux or any Unix or Linux platform that is certified
      with Java 7 (version 1.7.0_21)

Browser Support
	* For Kaazing clients, see the Release Notes for this product at
	  http://developer.kaazing.com/documentation/index.html
	* Command Center is fully supported by the following browsers:
		* Google Chrome version 26 or higher
		* Mozilla Firefox version 21 or higher
		* Microsoft Internet Explorer version 9 or higher
		* Apple Safari version 5 or higher
		* Android version 2.3 or higher

Starting Kaazing WebSocket Gateway
=============================================================================  

    To start Kaazing WebSocket Gateway, run the gateway.start command from the
    directory in which you have just unpacked the Kaazing WebSocket Gateway
    distribution, for example: C:\kaazing\kaazing-websocket-gateway-version\bin.

	For detailed instructions on starting the Gateway, see the "Setting Up"
	document for this edition at:
	
	http://developer.kaazing.com/documentation/index.html

  View the Demos and Documentation

    If you downloaded, installed, and started the full version of Kaazing
    WebSocket Gateway, you can view the demos by visiting:

	http://localhost:8001/
	
	For information about the product, see the documentation for this edition
	at http://developer.kaazing.com/documentation/index.html
