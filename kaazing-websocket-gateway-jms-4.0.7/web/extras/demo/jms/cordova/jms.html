Cordova JMS Demo

You can use the JMS JavaScript API with Apache Cordova to build native iOS and Android applications using HTML, CSS and JavaScript. The included demo helps you build your own JavaScript based iOS or Android applications using Apache Cordova. The demo folder contains scripts to copy a JavaScript JMS demo application into your iOS or Android development environment. The scripts and source code for this demo are located in the GATEWAY_HOME/demo/cordova directory of this release. For more information, see Walkthrough: Deploy a JMS JavaScript App as a Hybrid iOS App and Walkthrough: Deploy a JMS JavaScript App as a Hybrid Android App.

This demo uses a JMS client library to send and receive messages over the Web with a JMS-compliant message broker.
