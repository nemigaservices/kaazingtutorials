Microsoft Excel JMS Demo


Click to enlarge

This demo showcases integration between Kaazing Gateway and Microsoft Excel. In this example, stock prices are streamed live, in real-time, into an Excel spreadsheet. All of the source code is included so you can see how it works, and get started on your own application.

The spreadsheet calls out to an RTD server, which is a DLL compiled from Visual C#. The RTD server uses the .NET client to connect to Kaazing Gateway and subscribes to a topic that is broadcasting stock prices.

Look in GATEWAY_HOME/demo/excel/jms for the zip with an installer for this demo. The demo only runs on Windows because Microsoft Excel only supports calling an RTD server on Windows. The installer will install the Excel spreadsheet, source code, and also register the RTD server DLL.
