Silverlight JMS Demo

The demonstration on this page shows a Silverlight-based JMS client that communicates via WebSocket with a JMS-compliant message broker. Source code for this demo is located in the GATEWAY_HOME/demo/silverlight/src/jms directory of this release.

This demo Silverlight application uses a JMS client to send and receive messages over WebSocket. To learn how to build a Silverlight client using the JMS client libraries, see Checklist: How to Build Microsoft Silverlight and .NET Clients.

Run the demo

Microsoft Silverlight Client Requirements

This demo requires the Microsoft Silverlight plugin version 4.0 or higher running in a certified browser and operating system. Currently, the Microsoft Silverlight client library is not certified on:

  • Google Chrome 4 for Mac OS X
  • Safari on Windows
  • Linux

Your environment does not appear to meet the requirements. You can still attempt to run the demos by visiting the links below.
