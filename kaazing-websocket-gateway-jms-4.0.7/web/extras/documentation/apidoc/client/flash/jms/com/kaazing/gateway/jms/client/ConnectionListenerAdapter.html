Packagecom.kaazing.gateway.jms.client
Classpublic class ConnectionListenerAdapter
ImplementsConnectionListener

The ConnectionListenerAdapter is a convenience call to create and pass a ConnectionListener to set connection.connectionListener. This convenience class implements the ConnectionListener interface with individual named or anonymous functions. Each function is called with a single argument upon completion of connect, start, stop, and close. Example: function (conn:Connection):void { trace("CONNECTED"); }
