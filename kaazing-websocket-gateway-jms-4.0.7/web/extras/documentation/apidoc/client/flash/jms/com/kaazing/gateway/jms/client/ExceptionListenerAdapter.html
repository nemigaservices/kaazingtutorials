Classpublic class ExceptionListenerAdapter
ImplementsExceptionListener

The ExceptionListenerAdapter is a convenience call to create and pass a ExceptionListener to set connection.exceptionListener. This convenience class implements the ExceptionListener interface with an individual named or anonymous function. Example: function (e:Error):void { trace(e.message); }
