Classpublic class MessageListenerAdapter
ImplementsMessageListener

The MessageListenerAdapter is a convenience call to create and pass a MessageListener to set consumer.messageListener. This convenience class implements the MessageListener interface with an individual named or anonymous function. Example: function (msg:Message):void { trace(message.messageId+" received"); }
