InheritanceVoidFuture com.kaazing.gateway.jms.client.GenericFuture

Future-based support for asynchronous functions returning void. Methods that would block instead accept a callback which is called when its execution completes asynchronously. Once the callback is invoked, call checkError() to ensure that no errors occurred during processing.
