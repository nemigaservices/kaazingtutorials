public function StompConnectionFactory(providerURL:String, properties:StompConnectionProperties = null)

Creates a new StompConnectionFactory to create connections to the target url specified. A WebSocket is used to establish the connection. The url should specify a protocol such as ws: or wss:. For example: ws://localhost:8000/jms

public function createConnectionWithClientID(username:String = null, password:String = null, clientID:String = null, callback:Function = null):ConnectionFuture

Creates a JMS Connection via WebSocket, using the username and password specified as credentials to authenticate with the Gateway, and using the given clientID to distinguish the durable subscribers for this connection from those of other users.

Parameters
username:String (default = null) — Client identifier. This is combined with durable subscriber names, using the format given in the stomp.jms service parameter " client.durable.name.format", to make them unique for the current user
