Method detail
lookup()method
public function lookup(name:String):Object

Retrieves the named object. Throws an Error if the name is not bound. If name is "ConnectionFactory", returns a ConnectionFactory object that connects to the provider url (Gateway url) defined in the env properties passed to the StompInitialContext() constructor. Returns a Destination representing a topic or queue in the JMS provider. The following format is used for all topics and queues, regardless of the server format.
/topic/destination (Topic)
/queue/destination (Queue)

Use the Session interface to create Temporary Topics and Queues.
