Class JmsConnectionProperties

The class defines the JMS Connection properties. 


Example Usage - Configure maximum reconnect attempts to 5
var connectionProperties = new JmsConnectionProperties();
connectionProperties.reconnectAttemptsMax = 5;

// connectionProperties should be provided while creating JmsConnectionFactory object
var connectionFactory = new JmsConnectionFactory(url, connectionProperties);
